📡 Custom Events đã tích hợp
quiz_start
Gửi khi user bấm "Vào làm bài" và bài thi bắt đầu. Dùng để đếm số lượt thi theo module.
module_id — ID kỹ thuật (VHF, Radar...)
module_name — Tên đầy đủ của module
vi_tri — Long Thành / Tân Sơn Nhất
timestamp — ISO 8601
quiz_complete
Gửi khi nộp bài (bấm nút hoặc hết giờ). Dùng để phân tích điểm số và tỷ lệ đạt/không đạt.
module_id / module_name
score_percent — 0–100
correct_answers — Số câu đúng
total_questions — Tổng số câu
result — "pass" / "fail"
auto_submit — true nếu hết giờ
quiz_abandon
Gửi khi user đóng tab / rời trang trong khi đang thi. Dùng để tính tỷ lệ bỏ giữa chừng.
module_id / module_name
question_reached — Câu đang làm (1-based)
questions_answered — Số câu đã trả lời
time_remaining — Giây còn lại
